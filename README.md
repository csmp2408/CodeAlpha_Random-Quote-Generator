# Random Quote Generator

A modern and responsive Random Quote Generator web application built using HTML, CSS, JavaScript, Node.js, and Express.js.

## Features

- Generate random quotes instantly
- Display quote text and author
- New Quote button
- Prevent duplicate consecutive quotes
- Copy quote to clipboard
- Share quote functionality
- Quote view counter
- Dark mode support
- Responsive design
- Clean and modern UI

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Node.js
- Express.js
